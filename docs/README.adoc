Asciidoctor 是一个比 Markdown 更先进、复杂的文档书写和生成工具，它定义了类似 Markdown 的文字格式，但是能生成更好看的文档、提供
更强大的排版、内容创作功能，尤其是在文档中加入示例代码的方法也更为方便。

下面让我们开始Asciidoc使用之旅吧...
